Virtual Assistant

Command Description
answer [QUESTION] Wolfram Alpha: Answer summary
article [KEYWORDS] Wikipedia: One sentence explanation
weather [LOCATION] OpenWeatherMap: Description and temperature
dictionary [WORD] Oxford Dictionary: Word definition
video [KEYWORDS] YouTube: Embedded video
map [LOCATION] Google Maps: Embedded map
translate [TARGET_2LETTER] [TEXT] Google Translate: Translate text
search [INPUT] New tab Google search
webcam Face: Analyze age, gender, expressions
location MapQuest: Current location
time Current time
date Current date
joke Joke: Random
